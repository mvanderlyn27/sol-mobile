# Slice of Life mobile + backend

## How to start

- downlaod bun
  - curl -fsSL <https://bun.sh/install> | bash
- download packages
  - bun install
- setup env vars
  - create .env, .env.production, .env.development, .env.test
  - get values from michael
- start project
   bun start:test

## Local DB

- go to /supabase follow readme
- start project with local db
  - bun start
